Portfolio

SpartaHack - SpartaNow

SpartaNow is a mobile application for Andriod, iOS, and the web. It is designed to help users determine whether it would be faster to walk or use the Cata bus system on MSU's campus to get from one location to another. This was a collaborate project that was completed over a weekend for MSU's Hackathon, SpartaHack. The project written with Java for the Android App, Swift for the iOS app, and HTML5 for the web app. PHP was used for all backend calls.

MSU CSE Capstone - Brand Loyalty

Brand Loyalty is a tablet application developed for Urban Science by the Urban Science MSU Capstone Team, myself and four others. The project's main goal was to create an application that allowed users to quickly assess how a geographic area's KPIs (Key Performance Indicators) were doing in relation to a larger geographic region and in comparison to any competitors. The application was created using AngularJS and HTML5 for the frontend and PHP for the backend.

CSE 480 - Final Project

The final project of CSE 480 was to create a site where users could collaboratively work on shared documents. Users were able to create projects, add documents to those projects, search and add friends, and invite users/friends to collaborate on thier projects. Every time a user modifies a document, a new document is created and added to a tree that shows the document history. Any of the files seen in the tree can be edited, which will create a new version of that file. This was a team project coded in HTML5 for the frontend and PHP for the database calls.
